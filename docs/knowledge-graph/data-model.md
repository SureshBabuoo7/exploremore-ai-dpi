# Tourism Knowledge Graph Data Model

## Entity Structure

Each entity contains:

- Entity ID
- Name
- Category
- Description
- Latitude
- Longitude
- District
- Tags
- Verification Status

## Relationship Structure

Each relationship contains:

- Source Entity
- Target Entity
- Relationship Type
- Confidence Score
- Evidence Source

## Circuit Structure

Each circuit contains:

- Circuit ID
- Circuit Name
- Theme
- Included Entities
- Estimated Duration
- Recommended Sequence
